# Quick Start - Anesthesia Calculator Pro

1. **Start Backend**: Launch XAMPP/WAMP and ensure PHP/MySQL is active.
2. **Database**: Import `schema.sql` into a new DB named `anesthesia_calculator`.
3. **API**: Host the `backend` folder.
4. **Build App**: Open the `aenesthesia-calculator-pro-main` folder in Android Studio and build the project.
5. **Install**: Run the app on an emulator or physical device.

Check `MYSQL_SETUP_GUIDE.md` for detailed instructions.
